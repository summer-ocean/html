# スクロール

## HTML
```html
<div clas="table-wrapper">
  <table>
  </table>
</div>
```

## CSS
```css
.table-wrapper {
  overflow: scroll;
}
table {
  width: 幅;
  height: 高さ；
}
```

### 横
```css
.table-wrapper {
  overflow-x: scroll;
}
table {
  width: 幅;
}
```

### 縦
```css
.table-wrapper {
  overflow-y: scroll;
}
table {
  height: 高さ；
}
```
