\<a>
# 色
link → visited → hover → activeの順に書く

## 実践的なex
```css
a {
	color: #0073bc;
}
a:hover {
	color: #b7dbf2;
}
a:active {
	color: #4ca4e8;
}
```

## href属性がある
```css
a:link {
	color: 色;
}
```

## 訪問済み
```css
a:visited {
	color: 色;
}
```

## カーソルが乗ってる
```css
a:hover {
	color: 色;
}
```

## クリックしたとき
```css
a:active {
	color: 色;
}
```
