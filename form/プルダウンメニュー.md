# プルダウンメニュー
```html
<div>
	<label>
		題名<br>
		<select name="名前">
			<option value="値1">テキスト1</option>
			<option value="値2">テキスト2</option>
		</select>
	</label>
</div>
```

```html
<div>
	<label for="ID">題名</label><br>
	<select id="ID" name="名前">
		<option value="値1">テキスト1</option>
		<option value="値2">テキスト2</option>
	</select>
</div>
```
